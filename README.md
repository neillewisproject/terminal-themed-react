# Terminal Themed React App

A terminal-style portfolio application built with React + Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Available commands in UI

- `help`
- `about`
- `skills`
- `projects`
- `experience`
- `contact`
- `whoami`
- `date`
- `clear`
- `sudo`
